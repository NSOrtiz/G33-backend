# Koders API 1v generation javascript 33

This is a repo created during the backend module in [kodemia](https://kodemia.mx)

## How to run

1. install dependecies
```
nmp install
```
2. create an `.env` file
```
touch .env
```
3. to run in dev 
```
npm run dev
```
3. to run in production mode
```
npm start
```
